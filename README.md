# 📌 Recomendador de peliculas

En la actualidad, muchas personas tienen dificultades para elegir una película que ver, debido a la gran cantidad de opciones disponibles en internet. Plataformas como Netflix o páginas web muestran demasiadas opciones sin una organización clara basada en calidad o preferencia inmediata, lo que provoca que los usuarios pierdan tiempo buscando sin tomar una decisión rápida.

Además, no todos los usuarios desean navegar por múltiples páginas o aplicaciones, sino que prefieren una herramienta sencilla, rápida y directa que les permita encontrar películas según su género favorito o por nombre específico, y que además les muestre una referencia de calidad basada en calificaciones.

---

# 🎯 Sectores a los que va dirigido

La solución propuesta está enfocada a:

* Público en general (personas que desean ver películas)
* Estudiantes
* Usuarios de plataformas digitales
* Personas con poco tiempo para decidir qué ver

---

# 💡 Solución Propuesta

Se desarrolló una aplicación web utilizando Vue que permite a los usuarios buscar y filtrar películas de manera rápida, eficiente y organizada.

---

# ⚙️ Funcionalidades del Programa

La aplicación cuenta con las siguientes funciones:

* 🔍 **Búsqueda en tiempo real**:
  Permite al usuario escribir el nombre de una película y obtener resultados instantáneamente sin necesidad de recargar la página.

* 🎬 **Filtro por género**:
  El usuario puede seleccionar categorías como acción, comedia, terror, romance o ciencia ficción para visualizar únicamente películas de ese tipo.

* ⭐ **Orden por calificación**:
  Las películas se muestran ordenadas desde la mejor calificada hasta la de menor puntuación, facilitando la toma de decisiones.

* 📊 **Visualización en tabla**:
  Los resultados se presentan de forma clara y organizada, mostrando nombre, género y calificación.

* 🖱️ **Interfaz interactiva y dinámica**:
  Gracias a Vue, la página responde en tiempo real a las acciones del usuario, mejorando la experiencia de uso.

---

# ✅ Cómo resuelve el problema

Esta aplicación reduce el tiempo que una persona tarda en decidir qué película ver, ya que:

* Centraliza opciones recomendadas
* Permite búsquedas rápidas
* Filtra por preferencias específicas
* Muestra calidad de películas mediante calificaciones

De esta forma, el usuario puede tomar decisiones más rápidas y eficientes sin necesidad de navegar en múltiples plataformas.
